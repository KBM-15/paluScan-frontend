# 🩸 PaluScan - Frontend

**A medical-grade frontend for AI-powered malaria detection**

## 🎯 About

This is the complete frontend interface for **PaluScan**, an AI system that detects malaria parasites from blood smear images. The interface provides:

- 📸 Image upload (drag & drop supported)
- 🔬 Real-time AI analysis
- 📊 Confidence score visualization
- 🛡️ Malaria prevention guidelines
- 👩‍⚕️ Professional medical UI/UX

## 🔧 Technologies Used

- HTML5
- CSS3 (Medical-grade design system)
- JavaScript (ES6+)
- FastAPI backend integration
